Folders:
-data: contains data used for model development and validation
	-lr1865sz_2c_1_350_419.txt to lr1865sz_2c_4_350_419.txt: the charging curves sampled from 4 batteries cycled at a 2C charging rate, respectively.
	-lr1865sz_3c_1_350_419.txt to lr1865sz_3c_4_350_419.txt: the charging curves sampled from 4 batteries cycled at a 3C charging rate, respectively.
-models: contains trained models

Python files:
-2c_train.py: the python file for training models using 2C data.
-3c_train.py: the python file for training models using 3C data.
