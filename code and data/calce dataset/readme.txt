Folders:
-data: contains data used for model development and validation
	-CALCE_35.txt to CALCE_38.txt: the input data.

-models: contains trained models

-transferbasis: contains the pretrained model from the oxford dataset.
	-oxford_model.hdf5: A model pretrained based on the oxford dataset.

Python files:
-calce_direct_train.py: the python file for directly training a model.
-calce_transfer.py: the python file for training a model based on transfer learning.
